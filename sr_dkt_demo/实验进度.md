### 架构变更：衰减改残差门控（2026-06-18，model.py）
- **现象**：架构修复(λ独立+衰减上隐状态)后的新代码，SR-DKT AUC 反比旧代码低（旧 run AUC≈0.877，Constraint 恒 0）。
- **诊断**：旧版纯乘法 `h×exp(-λΔt)`（clamp 下限 0.01）会把整个隐状态压到 ~0，破坏进入预测的表征 → 掉 AUC。且 Δt 本就喂进 LSTM，显式衰减与之冗余，预测力主要来自 LSTM。
- **改法**：残差门控 `h×(β+(1-β)exp(-λΔt))`，系数 ∈ [β,1]；新增可学习保留底 `β_s`(高,Storage 持久) / `β_r`(低,Retrieval 易挥发)，差异化遗忘的第二维度。train.py 已加 β_s/β_r 日志。
- **影响**：旧 SR-DKT checkpoint(AUC 0.877)作废，需用残差门控**重跑**。期望：AUC 回到≥旧水平，且 λ/β 仍可解释。
- **定位**：相比 DKT-Forget(学习式融合时间特征、黑箱遗忘)，SR-DKT 保留显式 λ + 双状态 S/R + 差异化，可解释性是卖点。

## 五、研究定位与策略决策（2026-06-18）

- **目标解耦**：毕业(保底，**顶会非硬指标**)≠ 顶会(导师期望，冲刺项)。先焊死毕业那条线，再拿余力赌顶会。
- **创新口径**（关键，反复确认）：**不以"存储-提取双状态遗忘"为头条**(被 MemoryKT 2025 / CPF 2024 占)，**也不以"准确率最高"为卖点**(simpleKT/pyKT 封顶，FoLiBi 证明这类遗忘改进就 ~2% AUC)。改钉在 **"主动做题/被动看视频的行为分路 + 跨记忆类型差异化遗忘(λ_r>λ_s) + S/R 驱动的学习动作推荐(KT-Harness)"**。详见论文 md §11.8。
- **发表路线**：**主攻中档期刊**(Q2/Q3 SCI、CCF-B/C，如 KBS / ESWA / Education and Information Technologies)——期刊比顶会更吃"完整扎实多实验"，匹配本工作。**顶会(CCF-A)需另加尖刀**：干预式推荐 + 严谨离线反事实/off-policy 评估(把 §8.3 的"预期 ΔKS 自评"换成真 estimator)、或 LLM-Agent、或因果可解释。
- **"SR 加到别的模型(通用模块)"判定**：标准 KT 数据**不行**(无视频，Retrieval 路径空)；**MOOC 上可行**，但**必须配"X+视频单流"公平对照**证明是架构而非数据。**先验命门再扩张**：先跑 SR-DKT vs DKT+视频；成立才做 SR-DKVMN 等(只做 2 个骨干，跳过 Transformer，控成本)。
- **对导师(只看数据，但会查 baseline)**：主秀**消融大落差表 + 推荐层 HR/NDCG + 可解释可视化**(真实且好看)，AUC 对比表别当主秀；用领域证据(simpleKT/pyKT/FoLiBi)垫"领域提升普遍 1-2%"。
- **底线**：不造假；论文 §8.3 / §20 的"预期"数必须换成真实结果。
