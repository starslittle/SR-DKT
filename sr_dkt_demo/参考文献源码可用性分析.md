## SR-DKT 参考文献源码可用性分析

### 一、概述

本报告对你项目文档中引用的全部参考文献逐一排查了公开源码的可用性。核心结论是：知识追踪领域的主流经典模型（DKT、SAKT、AKT、DKVMN 等）都有成熟的开源实现，可以直接复用或通过 pykt-toolkit 统一调用；而你提到的几篇国内期刊论文和 2025-2026 年的新论文，绝大多数尚未公开源码。

---

### 二、有源码的论文

以下论文的模型实现可以通过 GitHub 直接获取：

#### 1. DKT — Piech et al., Deep Knowledge Tracing, NeurIPS 2015

DKT 作为知识追踪领域的开山之作，有多个社区复现版本。最推荐的是 pykt-toolkit 中的统一实现，此外还有独立的 PyTorch 复现项目。

- **pykt-toolkit**（综合 KT 工具包，包含 DKT 及多种变体）：https://github.com/pykt-team/pykt-toolkit
- **PyTorch 复现**：https://github.com/chsong513/DeepKnowledgeTracing-DKT-Pytorch
- **EduKTM**（另一个综合工具包）：https://github.com/bigdata-ustc/EduKTM

#### 2. SAKT — Pandey & Karypis, Self-Attentive Knowledge Tracing, 2019

SAKT 有作者官方实现，也包含在 pykt-toolkit 中。

- **作者官方代码**：https://github.com/shalini/SAKT
- **pykt-toolkit** 中也包含 SAKT 实现

#### 3. AKT — Ghosh et al., Context-Aware Attentive Knowledge Tracing, KDD 2020

AKT 有作者官方 GitHub 仓库，代码完整。

- **作者官方代码**：https://github.com/arghosh/AKT
- **pykt-toolkit** 中也包含 AKT 实现

#### 4. DKVMN — Dynamic Key-Value Memory Networks, 2017

DKVMN 作为经典记忆增强 KT 模型，在多个工具包中都有实现。

- **pykt-toolkit** 中包含 DKVMN
- **EduKTM** 中也包含 DKVMN 实现

#### 5. DKT-F — Nagatani et al., DKT with Forgetting, WWW 2019

DKT-F 没有独立的官方 GitHub 仓库，但你的项目中已经在 `baselines/baseline_models.py` 里自行实现了 DKT-F（使用 LSTMCell + 可学习 lambda 参数 + 逐步循环衰减）。此外 pykt-toolkit 中也有 DKT+Forget 的变体实现。

- **pykt-toolkit** 中有 `dkt_forget` 模型
- **你的项目** 中已自行实现

### 三、有统一工具包可调用的论文

#### pykt-toolkit（最推荐的综合工具包）

地址：https://github.com/pykt-team/pykt-toolkit

这是目前最完整的知识追踪开源工具包，由 pyKT 团队（Li et al., 2022）维护。它统一实现了以下模型，并提供标准化的数据预处理、训练和评估流程：

| 模型 | 是否包含 | 说明 |
|------|----------|------|
| DKT | 是 | 基础 LSTM 版 |
| DKT+ | 是 | DKT 改进版 |
| DKT-Forget | 是 | 遗忘增强版（对应 DKT-F） |
| DKVMN | 是 | 动态键值记忆网络 |
| SAKT | 是 | 自注意力版 |
| AKT | 是 | 上下文感知注意力版 |
| SAINT | 是 | 分离式注意力版 |
| GKT | 是 | 图知识追踪 |
| SKT | 是 | 模糊知识追踪 |
| ATKT | 是 | 对抗训练知识追踪 |
| DIMKT | 是 | 难度感知知识追踪 |
| simpleKT | 是 | 简单但强力的基线 |
| LBKT | **否** | 未包含行为导向 KT |
| KADKT | **否** | 未包含 |
| BEKT | **否** | 未包含 |

#### EduKTM（中国科学技术大学）

地址：https://github.com/bigdata-ustc/EduKTM

另一个综合工具包，包含 DKT、DKVMN、SAKT 等经典模型的实现。

### 四、没有找到公开源码的论文

以下论文在搜索时未找到公开的 GitHub 仓库或可复现的代码：

#### 1. LBKT / LB-KT — Learning Behavior-oriented Knowledge Tracing, KDD 2023

这是与你的 SR-DKT 最直接相关的竞争基线。LBKT 关注行为导向建模，发表在 KDD 2023。目前**没有找到作者官方公开的 GitHub 仓库**。不过你的项目中已经在 `baselines/baseline_models.py` 里自行实现了 LBKT（使用 MultiheadAttention + behavior_gate 结构），所以这个基线在你的实验中已经可用。

#### 2. KADKT — 融合学习者知识吸收能力的深度知识追踪模型，华南师范大学学报 2025

这是一篇国内期刊论文，将行为划分为显性/隐性行为。目前**没有找到公开源码**。论文链接：https://journal-n.scnu.edu.cn/article/doi/10.6054/j.jscnun.2025031

#### 3. BEKT — Behavior-Enhanced Knowledge Tracing, 2026

这是一篇较新的论文。目前**没有找到公开源码**。不过你的项目中已经在 `baselines/baseline_models.py` 里自行实现了 BEKT（使用 TransformerEncoderLayer + cross-feature + MLP 结构），所以这个基线在你的实验中已经可用。

#### 4. LFKT — 李晓光等，学习与遗忘融合的深度知识追踪模型，软件学报 2021

目前**没有找到作者官方公开的 GitHub 仓库**。

#### 5. DDKT — LLM-driven Dual-channel Difficulty Knowledge Tracing, arXiv 2025

目前**没有找到公开源码**。该论文使用 Transformer + LLM + RAG 架构，与你的 LSTM 双路径方案差异较大。

#### 6. MFKT — 多行为特征知识追踪，太原理工 2024

目前**没有找到公开源码**。

#### 7. CPF — 个性化遗忘, 2024

目前**没有找到公开源码**。

#### 8. MALPP — Multi-Agent Learning Path Planning via LLMs, arXiv 2026

目前**没有找到公开源码**。这是一篇 2026 年的新论文。

#### 9. MAIC / OpenMAIC — From MOOC to MAIC, 2025

目前**没有找到公开源码**。

#### 10. 其他中文论文

田雪涛博士论文（2022）、顾小清等综述（2024）、刘宇等综述（2022）、陈恩红团队可解释 DKT（2021）等——这些属于综述或博士论文，通常不提供可运行的代码仓库。

### 六、建议

**如果只是想跑基线对比实验**：你项目中的自行实现已经够用。如果希望验证自己的实现是否与原版一致，可以用 pykt-toolkit 中的标准化实现作为交叉验证（特别是 DKT、SAKT、AKT、DKVMN 四个模型）。

**如果想引入更多基线**：pykt-toolkit 是最推荐的扩展来源。它支持 ASSISTments 数据集的标准化预处理，与你的数据管道比较兼容。它额外包含了 SAINT、GKT、ATKT、DIMKT、simpleKT 等模型，都可以直接调用。

**如果需要复现 LBKT 或 BEKT 的原版代码**：这两篇论文目前都没有公开源码。你已经自行实现了它们，建议在论文中注明"自行复现"，并尽量对齐论文中描述的结构细节。
